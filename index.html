  4. Important CMD Commands
    1.commandname | clip(Copies command output to Clipboard)
2.color {color code} (changes the color of terminal acc. to color code ,, get color code by  "help color"  command)
3.wmic product get name(Lists all the installed programs on your computer)
4.press f7 Key to view Command history
5.md {folder name}\ (Creates folder with reserved names of windows which is generally not allowed)
6.telnet towel.blinkeneyes.nl(To watch StarsWars Movie in Command Prompt:) )
